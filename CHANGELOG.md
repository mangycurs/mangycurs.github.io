# История изменений

Все заметные изменения в проекте "Шавки" будут описаны в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.0.0] - 2026-05-09

### Добавлено
- Полноценный сайт про шавок с десктопной и мобильной версией
- Два языка (русский, английский) с переключением
- Три темы (светлая, тёмная, системная)
- Режим для слабовидящих с настройками шрифта, цветовых схем и синтезом речи
- PWA: Service Worker, манифест, офлайн-режим
- APK для Android (Trusted Web Activity)
- SEO-оптимизация: sitemap, robots.txt, Open Graph, Schema.org
- Фавиконки всех размеров
- Адаптивная вёрстка, таб-бар на мобилках
- Переключение между десктопным и мобильным видом (кнопка в шапке)
- Плавный скролл на десктопе

### Исправлено
- Ошибки загрузки картинок dog.ceo заменены на рабочие ссылки
- BVI-панель теперь корректно открывается/закрывается

### Удалено
- Ничего не удаляли, только добавляли.
