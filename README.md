# ШАВКИ — САМЫЙ ДЕРЗКИЙ САЙТ О СОБАКАХ 🐕‍🦺

[![Шавки лого](/assets/logo/Logo.png)](https://mangycurs.github.io)

**Резко. Беспощадно. Нагло.**  

Шавки – это сайт для тех, кто ценит настоящих собак. Ставки, породы, топ-события, а также полная поддержка слабовидящих, две темы, два языка и офлайн-режим. Проект создан, чтобы порвать интернет на части.

## 📱 Установка и использование

- 🌐 **Онлайн версия**: [mangycurs.github.io](https://mangycurs.github.io)
- 📲 **Android APK**: [Скачать последний релиз](https://github.com/mangycurs/mangycurs.github.io/releases/latest) (3 MB, TWA-обёртка, работает офлайн)
- 🧑‍🦯 **Версия для слабовидящих**: на сайте нажми на иконку очков
- ⚫ **Тёмная/светлая тема**: кнопка луны/солнца
- 🇷🇺🇺🇸 **Два языка**: русский и английский

## ✨ Функции

- Фулл PWA (Service Worker кэширует всё, офлайн-доступ)
- Адаптивный дизайн (десктоп + мобильная версия с таб-баром)
- Три режима темы: светлая, тёмная, системная
- Полная SEO-оптимизация (Open Graph, Schema.org, sitemap, robots.txt)
- Режим для слабовидящих (изменение шрифта, цветовых схем, синтез речи)
- Переключение между десктопной и мобильной вёрсткой (кнопка в шапке)
- Манифест и фавиконки всех размеров

## 🛠️ Стек технологий

- HTML5 / CSS3 (CSS Variables, адаптив, анимации)
- JavaScript (SPA-роутинг, динамический рендер)
- Service Worker (кэширование, офлайн)
- Web App Manifest (установка на экран)
- Node.js (Bubblewrap) – сборка APK
