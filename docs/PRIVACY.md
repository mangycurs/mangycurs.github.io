# Политика конфиденциальности

Наш сайт и приложение НЕ собирают:
- личные данные (имя, адрес, телефон)
- геолокацию (вы сами разрешаете, но мы не используем)
- историю браузера
- данные с устройства

Что может использоваться:
- Анонимная Яндекс Метрика – для статистики посещений.
- Service Worker – кэширует файлы для офлайн-доступа, не передаёт данные.

Если вы переходите по внешним ссылкам (например, на Telegram), то действует их политика конфиденциальности.
