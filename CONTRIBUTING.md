# Как помочь проекту Шавки

Спасибо, что хочешь внести вклад! Любая помощь приветствуется, даже если ты просто нашел опечатку.

## Открытие Issue

Прежде чем создать новый issue, убедись, что похожей проблемы нет в списке. Используй шаблон:

**Тип:** Баг / Улучшение / Вопрос  
**Описание:** Чётко опиши, что не так.  
**Воспроизведение:** Пошаговая инструкция (если баг).  
**Ожидаемое поведение:** Что должно происходить.  

## Pull Requests

1. Форкни репозиторий.
2. Создай новую ветку: `git checkout -b feature/какая-то-хуйня`
3. Внеси изменения.
4. Убедись, что сайт работает (можно локально или на GitHub Pages preview).
5. Отправь Pull Request, опиши, что и зачем.

## Гайд по стилю кода

- HTML: отступы 4 пробела, комментарии на русском или английском.
- CSS: используй CSS Variables, избегай `!important` (если только не ебёт).
- JS: ES6, стрелочные функции, понятные имена переменных (не `a1`, а `pageContent`).

## Лицензия

Внося вклад, ты соглашаешься, что твой код будет распространяться под лицензией MIT.

**Спасибо, что делаешь шавок лучше. Ебёна крест!**
